Сценарий 1: Создание нового чата
Название теста: Проверка создания чата

Цель тестирования: Убедиться, что пользователь может создать новый чат
и стать его первым участником.

Предпосылки:
- Пользователь зарегистрирован
- API сервер запущен

Шаги тестирования:
1. Отправить POST запрос на /api/chats с данными чата
   Ожидаемый результат: Статус 201 (Created)

2. Проверить тело ответа
   Ожидаемый результат: name = "Work Chat", members_count = 1

3. Проверить внутренние коллекции
   Ожидаемый результат: Чат добавлен в chats, создатель добавлен в members

Сценарий 2: Отправка сообщения
Название теста: Проверка отправки сообщения в чат

Цель тестирования: Убедиться, что участник чата может отправить сообщение.

Предпосылки:
- Существует чат с участниками
- Пользователь является участником чата

Шаги тестирования:
1. Отправить POST запрос на /api/messages с текстом сообщения
   Ожидаемый результат: Статус 201

2. Проверить данные сообщения
   Ожидаемый результат: text = "New message", is_read = False

3. Проверить обновление чата
   Ожидаемый результат: Сообщение добавлено в список messages чата

Сценарий 3: Получение сообщений чата
Название теста: Проверка получения истории сообщений

Цель тестирования: Убедиться, что участник может получить все сообщения чата.

Предпосылки:
- В чате есть несколько сообщений

Шаги тестирования:
1. Отправить GET запрос на /api/messages?chat_id={id}
   Ожидаемый результат: Статус 200

2. Проверить количество сообщений
   Ожидаемый результат: total = 2

3. Проверить содержание сообщений
   Ожидаемый результат: Присутствуют оба тестовых сообщения

Сценарий 4: Добавление участника в чат
Название теста: Проверка добавления нового участника

Цель тестирования: Убедиться, что администратор может добавить нового участника.

Предпосылки:
- Существует чат
- Существует пользователь, не состоящий в чате

Шаги тестирования:
1. Проверить отсутствие пользователя до добавления
   Ожидаемый результат: Пользователя нет в members

2. Отправить POST запрос на /api/chats/{id}/members
   Ожидаемый результат: Статус 200

3. Проверить добавление
   Ожидаемый результат: Пользователь добавлен в members и в его список чатов

Сценарий 5: Удаление участника из чата
Название теста: Проверка удаления участника

Цель тестирования: Убедиться, что администратор может удалить участника из чата.

Предпосылки:
- В чате есть несколько участников

Шаги тестирования:
1. Проверить наличие пользователя до удаления
   Ожидаемый результат: Пользователь есть в members

2. Отправить DELETE запрос на /api/chats/{id}/members
   Ожидаемый результат: Статус 200

3. Проверить удаление
   Ожидаемый результат: Пользователь удален из members и из его списка чатов


Сценарий 6: Отметка сообщения как прочитанного
Название теста: Проверка отметки о прочтении

Цель тестирования: Убедиться, что пользователь может отметить сообщение как прочитанное.

Предпосылки:
- В чате есть непрочитанное сообщение

Шаги тестирования:
1. Проверить статус до отметки
   Ожидаемый результат: is_read = False, read_by пуст

2. Отправить PUT запрос на /api/messages/{id} с user_id
   Ожидаемый результат: Статус 200

3. Проверить обновленный статус
   Ожидаемый результат: is_read = True, пользователь в read_by

Сценарий 7: Удаление сообщения
Название теста: Проверка удаления сообщения

Цель тестирования: Убедиться, что сообщение можно удалить из чата.

Предпосылки:
- В чате есть сообщение

Шаги тестирования:
1. Проверить наличие сообщения
   Ожидаемый результат: Сообщение есть в коллекциях

2. Отправить DELETE запрос на /api/messages/{id}
   Ожидаемый результат: Статус 200

3. Проверить отсутствие после удаления
   Ожидаемый результат: Сообщение удалено из всех коллекций


Сценарий 8: Удаление чата
Название теста: Проверка каскадного удаления чата

Цель тестирования: Убедиться, что при удалении чата удаляются все связанные данные.

Предпосылки:
- Существует чат с сообщениями и участниками

Шаги тестирования:
1. Проверить наличие чата и связанных данных
   Ожидаемый результат: Есть чат, сообщения, членства

2. Отправить DELETE запрос на /api/chats/{id}
   Ожидаемый результат: Статус 200

3. Проверить удаление всего связанного
   Ожидаемый результат: Чат, все его сообщения и членства удалены
