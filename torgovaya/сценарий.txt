Сценарий 1: Создание объявления о продаже
Название теста: Проверка создания нового товара

Цель тестирования: Убедиться, что продавец может создать объявление о продаже товара.

Предпосылки:
- Продавец зарегистрирован
- Существует категория товара
- API сервер запущен

Шаги тестирования:
1. Отправить POST запрос на /api/products с данными товара
   Ожидаемый результат: Статус 201 (Created)

2. Проверить тело ответа
   Ожидаемый результат: title = "MacBook Pro", price = 1999.99, quantity = 3

3. Проверить внутренние коллекции
   Ожидаемый результат: Товар добавлен в products, в список продавца и в категорию


Сценарий 2: Просмотр товара с отзывами
Название теста: Проверка получения товара со всеми отзывами

Цель тестирования: Убедиться, что при просмотре товара отображаются все отзывы.

Предпосылки:
- Существует товар с отзывами

Шаги тестирования:
1. Отправить GET запрос на /api/products/{id}
   Ожидаемый результат: Статус 200

2. Проверить данные товара
   Ожидаемый результат: title = "iPhone", rating = 5.0

3. Проверить наличие отзывов
   Ожидаемый результат: В ответе есть поле reviews с одним отзывом

4. Проверить увеличение просмотров
   Ожидаемый результат: views увеличился на 1


Сценарий 3: Редактирование товара
Название теста: Проверка обновления информации о товаре

Предпосылки:

Шаги тестирования:

2. Проверить обновленные поля
   Ожидаемый результат: title = "iPhone 15", price = 1099.99

3. Проверить сохранение в коллекции
   Ожидаемый результат: В products данные обновлены


Сценарий 4: Удаление товара с отзывами
Название теста: Проверка каскадного удаления товара и отзывов

Цель тестирования: Убедиться, что при удалении товара удаляются все связанные отзывы.

Предпосылки:
- Существует товар с отзывами

Шаги тестирования:
1. Проверить наличие товара и отзывов до удаления
   Ожидаемый результат: Товар и отзывы есть в коллекциях

2. Отправить DELETE запрос на /api/products/{id}
   Ожидаемый результат: Статус 200

3. Проверить удаление товара
   Ожидаемый результат: Товар удален из products

4. Проверить удаление отзывов
   Ожидаемый результат: Все отзывы к товару удалены

5. Проверить удаление из списков
   Ожидаемый результат: Товар удален из списков продавца и категории

Сценарий 8: Фильтрация товаров
Название теста: Проверка поиска и фильтрации товаров

Цель тестирования: Убедиться, что можно фильтровать товары по разным параметрам.

Предпосылки:
- В системе есть несколько товаров в разных категориях

Шаги тестирования:
1. Фильтр по категории
   Ожидаемый результат: Только товары указанной категории

2. Фильтр по цене (min_price, max_price)
   Ожидаемый результат: Товары в заданном ценовом диапазоне

3. Фильтр по продавцу
   Ожидаемый результат: Только товары указанного продавца
